Laptop-Guard for Kotebe University

LaptopGuard is a Flask-based web application designed to manage student laptop check-ins and check-outs. It tracks student information, including profiles and laptop images, and logs their library time. The app features an admin dashboard for managing student records, updating information, and viewing logs. It supports student registration, profile updates, and displays status changes in real-time, making it ideal for educational environments where laptop usage needs to be tracked.
